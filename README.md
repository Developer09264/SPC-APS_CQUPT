## Description

这是一个专门针对结构化程序设计课程题库的项目，意在提升题库使用体验

众所周知，学校的题库不仅界面十分复古，还存在很多的功能性问题，包括但不限于：
- 题目跳转问题
- 题库不会记录进度
- 错一次就要做对两次才算对

面对如此种种问题，我们知道，**该出手了**

在原来的~~石山~~基础上修修补补显然不是很好的解决方式，*我们不愿意修筑空中楼阁*

于是便做出了如此重要决定：**将答题的过程自动化，解放双手，将答题的时间还给各位同学**

重要声明：本项目仅作学习交流使用，学习是自己的事，请同学们自行权衡

## 关于这些文件

### userScript.js（建议使用这个）

封装的tampermonkey脚本，简化使用流程，推荐

简易使用教程：

1. 首先，你需要给浏览器安装油猴(TamperMonkey)插件
2. 复制[这里](./userScript.js)的代码
3. 打开油猴脚本控制台，新建用户脚本，删除原有内容并且粘贴刚才复制的代码，保存并启用
4. 打开题库的答题界面，右下角出现“ASP”按钮，即可开始使用

简易说明：

1. 准备好茶，短视频，网课，或者其您觉得比刷题更有意义的东西
2. 鼠标点击按钮后，按钮变成绿色，显示“运行中”，开始自动答题
3. 此时您可以喝茶，刷视频，刷网课等
4. 在您认为可以结束的时候，再次点击绿色按钮，即可停止脚本
5. 如果有任何出错的地方，欢迎提出Issue

脚本设计速度约为6.5秒每题，速度较慢，是为了预防校园网波动产生的影响。觉得慢可以手动调节脚本中的参数。

### uSee.js

这是目前可用的用于实现“自动解析题目，生成AI提示词，提供复制”功能的脚本

#### 简易使用方法

1. 打开题库，找到题目界面
2. 打开开发者工具，找到控制台Console
3. 选择名字叫“`CPractice.aspx`”的求值环境（上下文，执行上下文），将整段脚本粘贴进去，回车执行
4. 页面右下角会出现按钮，点击之后会获取当前页面的题目，组成提示词，按照说明复制，点击周边空白处关闭弹窗
5. 答题完成后可以循环上一步操作
